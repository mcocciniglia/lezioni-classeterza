Cheat sheet – HTML base

Riferimento essenziale (manuale) per: struttura pagina, testo, entità, liste, link, tabelle.

⏱️ Consultazione durante la prova • Solo HTML

1) Struttura minima di una pagina

<!DOCTYPE html>
<html lang="it">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>Titolo</title>
</head>
<body>

</body>
</html>

2) Testo

  • <h1> titolo principale
  • <h2> titolo di sezione
  • <p> paragrafo
  • <strong> enfasi forte
  • <em> enfasi
  • <br> a capo
<h1>Titolo</h1>
<h2>Sezione</h2>
<p>Testo con <strong>strong</strong> e <em>em</em>.<br>Seconda riga.</p>

3) Entità (caratteri speciali)

  • &lt; → <
  • &gt; → >
  • &le; → ≤
  • &ge; → ≥
<p>2 &lt; 3, 3 &ge; 3, testo: &lt;esempio&gt;</p>

4) Liste

  • <ul> elenco puntato
  • <ol> elenco numerato
  • <li> elemento elenco
<ul><li>A</li><li>B</li></ul>
<ol><li>1</li><li>2</li></ol>

5) Link

  • <a> collegamento
  • href destinazione
  • target="_blank" nuova scheda
  • rel="noopener" sicurezza (con target blank)
<a href="https://esempio.it">Testo link</a>
<a href="https://esempio.it" target="_blank" rel="noopener">Nuova scheda</a>

6) Tabelle

  • <table> tabella
  • <thead>, <tbody>, <tfoot> sezioni
  • <tr> riga
  • <th> cella intestazione
  • <td> cella dati
  • colspan unisce colonne
  • rowspan unisce righe
<table border="1">
  <thead><tr><th>A</th><th>B</th></tr></thead>
  <tbody><tr><td>1</td><td>2</td></tr></tbody>
  <tfoot><tr><td colspan="2">Nota</td></tr></tfoot>
</table>
<tr><td rowspan="2">X</td><td>R1</td></tr>
<tr><td>R2</td></tr>

Checklist

  • Documento completo: doctype, html, head, body
  • Tag chiusi correttamente
  • Entità corrette per < e >
  • Liste: ul/ol contengono li
  • Tabella: tr contiene th/td; attenzione a rowspan/colspan
