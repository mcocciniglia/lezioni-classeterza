Selettori CSS: le regole principali

In CSS, il selettore dice “a quali elementi HTML applicare” una regola di stile. Sotto trovate le regole più usate, tradotte e con esempi.

⏱️ 1 ora • Classe: 4ª IT • Modalità: laboratorio

Idea chiave

Una regola CSS ha questa forma: selettore { proprietà: valore; } 

p {
  color: blue;
}

Significa: “tutti i paragrafi <p> diventano blu”.

Tabella: selettore → significato → esempio

Selettore Significato Esempio
Universale Si applica a tutti gli elementi della pagina. 
* { }

Seleziona tutti gli elementi.
Di tipo Seleziona elementi in base al nome del tag (p, h1, a, …). 
h1, h2, h3 { }

Seleziona tutti gli <h1>, <h2> e <h3>.
Classe Seleziona gli elementi con una certa class (preceduta da .). Una classe può essere usata su più elementi. 
.note { }

Seleziona tutti gli elementi con class="note".

p.note { }

Seleziona solo i <p> con class="note".
ID Seleziona l’elemento con un certo id (preceduto da #). In una pagina, l’ID dovrebbe essere unico. 
#introduction { }

Seleziona l’elemento con id="introduction".
Figlio diretto Seleziona un elemento che è figlio diretto di un altro (simbolo >). 
li > a { }

Seleziona gli <a> direttamente dentro un <li>.
Discendente Seleziona un elemento che si trova da qualche parte dentro un altro (non per forza figlio diretto). 
p a { }

Seleziona tutti gli <a> che stanno dentro un <p> (anche se in mezzo ci sono altri tag).
Fratello adiacente Seleziona l’elemento che è subito dopo un altro (simbolo +). 
h1 + p { }

Seleziona il primo <p> immediatamente dopo un <h1>.
Fratelli generali Seleziona tutti i “fratelli” che vengono dopo un elemento (simbolo ~), non per forza immediatamente. 
h1 ~ p { }

Seleziona tutti i <p> fratelli di <h1> che stanno dopo di lui.
